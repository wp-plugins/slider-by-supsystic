== Installation ==

First time Slider by Supsystic user

Thank you for choosing Slider by Supsystic! In order to create your slideshow, simply activate the plugin and click on "New Slider" in your left navigation menu. Once the Slider by Supsystic page loads, set the options for your responsive slider (height, images, transition effects etc). Copy and paste the shortcode into your post or page and Slider by Supsystic plugin will automatically create the slider on your page using the settings you chose above.

To install a plugin via FTP:
1. Download Slider by Supsystic .zip
2. Unarchive slideshow plugin
3. Copy the folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the plug-ins folders and files in the folder
6. Go to admin panel => open item "Plugins" => activate Slider by Supsystic plugin you need
